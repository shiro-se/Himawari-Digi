# Himawari Digital

Himawari Digital is a modern, single-page application (SPA) web development agency website built with vanilla JavaScript and Tailwind CSS. It features smooth animations, scroll-triggered reveals, and a dynamic, interactive layout.

## Features

- **Modern Single-Page Architecture**: Fast navigation without full page reloads using vanilla JavaScript.
- **Smooth Animations**: All pages feature premium scroll-triggered animations and effects.
- **Interactive 3D Cards**: The Services page uses JavaScript to create a rotating 3D carousel effect for service cards.
- **Reusable Components**: Clean separation of HTML structure and JavaScript logic.
- **Responsive Design**: Fully adaptive layout for desktop, tablet, and mobile devices.
- **Dark Mode**: Built-in support for system and manual theme toggling.

## Getting Started

### Prerequisites

- A modern web browser supporting ES6.
- [Node.js](https://nodejs.org/) and [npm](https://www.npmjs.com/) (optional, for running a local server).
- [Tailwind CLI](https://tailwindcss.com/docs/installation) (optional, for compiling CSS).

### Installation

No installation is required. This is a static site.

### Running Locally

**Option 1: Using Node.js (Recommended)**
1.  Open your terminal in the project directory.
2.  Run: `npx serve`
3.  Open [http://localhost:3000](http://localhost:3000) in your browser.

**Option 2: Using PHP**
1.  Open your terminal in the project directory.
2.  Run: `php -S localhost:8000`
3.  Open [http://localhost:8000](http://localhost:8000) in your browser.

**Option 3: Using VS Code Live Server**
1.  Open the project in VS Code.
2.  Install the [Live Server](https://marketplace.visualstudio.com/items?itemName=ms-vscode.live-server) extension.
3.  Right-click `index.html` and select "Open with Live Server".

## Usage

- Click any link in the header to navigate between pages.
- Scroll down the page to see animations trigger.
- Click the sun/moon icon in the header to toggle dark mode.
- Use the "hamburger" menu on mobile to open the navigation drawer.
